## 1.3 빌드 및 배포
### **빌드 아키텍처**
- git clone 이후 .env파일 루트에 생성
- docker compose up --build를 통해 실행
- localhost:3000으로 접근

### **배포 아키텍처**

1. **개발자가 release 브랜치에 push**
2. 젠킨스가 git clone 워크스페이스에 복사
3. 젠킨스 파일을 읽고 credential 환경변수에 주입
4. 도커 컴포즈 빌드
5. 로드 실패시 과거 성공버전으로 배포
